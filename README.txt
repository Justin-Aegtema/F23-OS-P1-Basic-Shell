BASIC UNIX SHELL MADE FROM SCRATCH

Built in commands:
- exit - exits program
- cd - change to directory user specifies
- path - user input to select path for future commands

Features:
- redirect command output to file with ">"
- input multiple commands in parallel with "&"
- can input using a .txt file (batch mode) or with user typed commands



Try inputting the test cases in /xtratestcases !

INPUT .TXT FILE:
prompt> ./dash batch.txt

NO INITIAL INPUT GIVES PERSISTENT PROMPT:
prompt> ./dash
dash>

Commands to try in linux:
- echo, ls, pwd
dash> echo
dash> ls
